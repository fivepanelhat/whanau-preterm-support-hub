## Why archive?

`whanau-preterm-support-hub` was an early Next.js scaffold (v0.1.0) with the same vision as **Front_Line_Whanau**. Dual public repos confused contributors and diluted focus.

Unique assets (release preflight + ADR) were ported into Front_Line_Whanau. Open issues should be re-filed there if still relevant.
